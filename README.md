# Royal Dry Cleaners - Professional Dry Cleaning Website

A modern, responsive website for Royal Dry Cleaners, a premium dry cleaning and laundry service business offering professional cleaning, alterations, and convenient pickup/delivery services.

## Features

- **Responsive Design**: Works seamlessly on all devices (desktop, tablet, mobile)
- **Modern UI**: Clean, professional interface with smooth animations and transitions
- **Service Showcase**: Comprehensive display of all cleaning services offered
- **Pricing Transparency**: Clear pricing information for all service categories
- **Contact Form**: Easy way for customers to get in touch
- **Mobile-Friendly**: Fully responsive navigation and layout
- **Interactive Elements**: Smooth scrolling, animations, and form validation
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## Technologies Used

- **HTML5**: Semantic markup for better SEO and accessibility
- **CSS3**: Modern styling with CSS variables, flexbox, and grid
- **JavaScript (ES6+)**: Interactive elements and animations
- **Font Awesome**: Icon library for visual elements
- **Google Fonts**: Clean, modern typography

## How to Use

1. Clone this repository
2. Open `index.html` in a web browser
3. Replace placeholder images with your actual images:
   - `logo.png` - Your company logo
   - `hero.jpg` - Hero section background image
   - `about.jpg` - About section image
4. Update contact information in the HTML file:
   - Phone numbers
   - Email addresses
   - Physical address
   - Business hours
5. Customize colors and content as needed:
   - Modify the CSS variables in `styles.css`
   - Update service descriptions
   - Adjust pricing information

## Customization

### Colors
Modify the CSS variables in `styles.css` to match your brand:
```css
:root {
    --primary-color: #2c5f7c;    /* Main brand color */
    --secondary-color: #48afdb;  /* Accent color */
    --accent-color: #f4b942;     /* Highlight color */
    --dark-color: #1a2332;       /* Dark text */
    --light-color: #f8f9fa;      /* Light background */
}
```

### Services
Edit the services section in `index.html` to reflect your actual offerings:
- Add or remove service cards
- Update service descriptions
- Change icons as needed

### Pricing
Update the pricing section with your actual pricing:
- Modify price ranges
- Add or remove pricing categories
- Update pricing notes

### Contact Information
Update the contact section with your business details:
- Phone numbers
- Email addresses
- Physical address
- Business hours
- Social media links
