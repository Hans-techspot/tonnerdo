# Amex Cleaners - Professional Dry Cleaning Website

A professional, responsive website for Amex Cleaners, a dry cleaning business offering premium cleaning services.

## Features

- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Modern UI**: Clean, professional interface with smooth animations
- **Service Showcase**: Display of all cleaning services offered
- **Contact Form**: Easy way for customers to get in touch
- **Mobile-Friendly**: Fully responsive navigation and layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome (for icons)

## File Structure

```
/
├── index.html          # Main HTML file
├── styles.css          # Main stylesheet
├── script.js           # JavaScript for interactivity
├── logo.png            # Company logo
├── hero.jpg            # Hero background image
├── about.jpg           # About section image
└── README.md           # Project documentation
```

## How to Use

1. Clone this repository
2. Open `index.html` in a web browser
3. Replace placeholder images with your actual images
4. Update contact information in the HTML file
5. Customize colors and content as needed

## Customization

- **Colors**: Modify the color scheme in `styles.css` (primary color is `#2c5f7c`)
- **Content**: Update text in `index.html` to match your business information
- **Images**: Replace placeholder images with your own high-quality images
- **Services**: Edit the services section to reflect your actual offerings
