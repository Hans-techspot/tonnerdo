# Amex Cleaners - Professional Dry Cleaning Website

A professional, responsive website for Amex Cleaners, a dry cleaning business offering premium garment care services.

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop devices
- **Modern UI**: Clean, professional interface with smooth animations
- **Service Showcase**: Displays all cleaning services with icons and descriptions
- **Pricing Information**: Clear pricing for different garment types
- **Contact Form**: Easy way for customers to get in touch
- **Smooth Navigation**: Smooth scrolling between sections
- **SEO Optimized**: Proper meta tags for search engine visibility

## Technologies Used

- HTML5
- CSS3 (with Flexbox and CSS Grid)
- Vanilla JavaScript

## How to Use

1. **View the Website**: Open `index.html` in any modern web browser
2. **Customize**: Edit the HTML and CSS to match your brand colors, services, and contact information
3. **Deploy**: Upload to any web hosting service (Netlify, Vercel, GitHub Pages, etc.)

## Customization

### Branding
- Update the logo text and colors in the CSS
- Change the hero image by modifying the background URL in the CSS
- Update the color scheme to match your brand

### Services
- Edit the service cards in the "Our Services" section
- Add or remove services as needed

### Pricing
- Update the pricing cards with your actual rates
- Modify the pricing section to reflect your service tiers

### Contact Information
- Update the contact form action to point to your email server
- Change the address, phone, and email in the contact section
- Update social media links

## Deployment

### GitHub Pages
1. Push to a GitHub repository
2. Go to Settings → Pages
3. Select main branch
4. Your site will be live at `https://username.github.io/repository`

### Netlify
1. Drag and drop the folder to Netlify
2. Or connect your GitHub repository
3. Site will be deployed automatically

### Vercel
1. Install Vercel CLI: `npm install -g vercel`
2. Run: `vercel`
3. Follow the prompts
